# N8N Automation Workflows

A collection of AI-powered automation workflows built using N8N.

## Workflows Included

### 1. Appointment Booking Bot
- AI chatbot that books appointments automatically
- Integrated with Google Calendar
- Collects name, contact, date, time from user

### 2. Google Calendar Agent
- Creates and fetches events from Google Calendar
- Uses OpenAI for natural language understanding
- Supports simple memory for context

## Tech Stack
- N8N (Automation platform)
- OpenAI GPT (AI responses)
- Google Calendar API
- Simple Memory (session context)
